2. What Data We Collect

2.1. User Profile Data

The user may enter the following data in the Application settings:

  • Name
  • Email address
  • Company name
  • Company address
  • Phone number

This data is stored locally on the device within the Application's storage and is included in exported PDF and DXF files. The data is not transmitted to the Company's servers.

2.2. Analytics Data

We use Firebase Analytics and Firebase Crashlytics to collect anonymous data about the use of the Application and to ensure its stability:

  • Device type, model, and operating system version
  • Language and regional settings
  • Actions within the Application (screen views, feature usage)

2.3. Crash Data (Crashlytics)

Firebase Crashlytics automatically collects the following information when a crash or error occurs in the Application:

  • Stack traces and exception information
  • Crashlytics installation ID (a unique UUID for the app instance)
  • Device model, OS version, and screen orientation
  • Amount of RAM and disk space
  • Device state at the time of the crash (charging, background/foreground)

Crashlytics does not collect personal data that can identify the user.

2.4. User Content

The Application stores data you create:

  • Floor plans and measurements
  • Wall elevations
  • Engineering point markers (electrical, water, sewage, radiators)

2.5. Voice Input (RECORD_AUDIO)

The Application requests the RECORD_AUDIO permission for voice-controlled drawing. The audio stream is transmitted to Google Speech Recognition servers for speech recognition. Audio is not saved by the Application. The user can opt out of voice input and use manual input instead.

2.6. Google Play Billing (Purchases)

The Application offers subscriptions and one-time purchases through Google Play. Google processes payment data — the Application does not receive bank card details. The Application receives from Google: subscription status, purchase history, and purchase tokens. Learn more: Google Play Policy.

2.7. System Data

The Application reads the installation time to calculate the trial period (30 days) and the country/region from the device's system locale to determine access conditions.

2.8. Data Export

Exported PDF and DXF files contain the user's personal data (name, email, company, address, phone number) as specified in the profile. When sharing files through third-party applications (e.g., email, messengers), the user is solely responsible for their distribution.

3. Purposes of Data Processing

We process your data for the following purposes:

  • Providing and supporting the functionality of the Application
  • Analyzing usage and improving the Application
  • Identifying and resolving technical errors
  • Processing purchases and subscriptions through Google Play
  • Communicating with users (support, updates)
  • Fulfilling legal obligations

4. Legal Basis for Processing

Personal data processing is carried out on the following grounds in accordance with the GDPR:

  • Consent (Art. 6(1)(a) GDPR) — for analytics
  • Performance of a contract (Art. 6(1)(b) GDPR) — for service provision, voice input, and purchase processing
  • Legitimate interest (Art. 6(1)(f) GDPR) — for improving the Application and ensuring security

5. Data Sharing with Third Parties

We may share data with the following categories of recipients:

  • Google LLC — as part of using Firebase Analytics, Firebase Crashlytics, Google Speech Recognition, and Google Play Billing (servers may be located outside the EEA; transfers are carried out based on Standard Contractual Clauses)
  • Government authorities — in cases provided for by law

We do not sell your personal data to third parties.

6. Data Retention

We retain your personal data for as long as necessary for the purposes described in this Policy:

  • User profile data — stored locally on the device until the Application is uninstalled by the user
  • Analytics data — up to 26 months (default Firebase Analytics settings)
  • Crash data (Crashlytics) — up to 90 days
  • Purchase data — in accordance with Google Play policy
