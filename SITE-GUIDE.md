# Руководство по сайту ClewByte.com

## Хостинг

Сайт размещён на **GitHub Pages** (не GitHub Packages — это другой сервис).

- **Репозиторий:** приватный, на бесплатном плане GitHub
- **GitHub Pages** раздаёт файлы из **корня** ветки `main`
- Настройки Pages заблокированы для изменения (ограничение бесплатного плана для приватных репо), но текущая конфигурация работает
- **Важно:** не менять настройки Pages в GitHub, не переключать Source — обратно настроить не получится

## Как обновлять сайт

### 1. Редактируешь файлы в `src/`

- Тексты → `src/i18n/en.json` и `src/i18n/ru.json`
- Разметка → `src/templates/`
- Стили → `src/css/style.css`
- Картинки → `src/images/`

**Никогда не редактируй файлы в корне** (`index.html`, `css/`, `js/` и т.д.) — они перезаписываются при сборке.

### 2. Коммитишь и пушишь

```bash
git add .
git commit -m "Описание изменений"
git push
```

При коммите автоматически срабатывает pre-commit hook:
- Запускает `npm run build`
- Собранные файлы добавляются в коммит

### 3. GitHub Pages обновляет сайт

Через 1–3 минуты после пуша изменения появятся на clewbyte.com.

## Фиксированные URL

Ссылки на политику конфиденциальности зафиксированы в Play Store:

| Язык | URL |
|------|-----|
| RU | `https://clewbyte.com/privacy.html` |
| EN | `https://clewbyte.com/privacy-en.html` |

Эти пути заданы в `build-config.js` → `urlOverrides` и не должны меняться.

## Добавление нового языка

1. Создать файл перевода `src/i18n/xx.json` (скопировать `en.json`, перевести)
2. Если нужна отдельная политика — создать `src/content/privacy.xx.html`
3. Запустить `npm run build` — новый язык подхватится автоматически
4. Закоммитить и запушить

## Частые ошибки

- **Редактировал корневые HTML файлы** — изменения пропадут при следующем билде. Редактируй только `src/`.
- **Сайт не обновился после пуша** — подожди 3 минуты, GitHub Pages кэширует.
- **Pre-commit hook не работает** — выполни `git config core.hooksPath .githooks`.
- **Не трогай настройки GitHub Pages** — текущая конфигурация (корень / main) работает, но заблокирована для изменений из-за приватного репо на бесплатном плане.
